Metodología

El sistema se basa en la ingesta automatizada de datos económicos, procesados mediante scripts Python ejecutados en ciclos programados. Los datos se almacenan en formato JSON y se transforman en indicadores clave.

Estructura del sistema

Pipeline modular:

  • Captura de datos (scripts Python)
  • Procesamiento (indicadores)
  • Generación HTML estático
  • Publicación automática en GitHub Pages

Modelo OSINT

El Radar OSINT se basa en correlación de variables económicas para detectar escenarios de riesgo. La combinación de IPC y energía permite anticipar tensiones macroeconómicas.
